# 下载方式

可以使用 git bash 直接 clone 本厂库。
```bash
git clone https://github.com/weijiew/web-step.git
```

也可以直接复制下面这个链接，直接下载。
```bash
https://codeload.github.com/weijiew/web-step/zip/master
```
